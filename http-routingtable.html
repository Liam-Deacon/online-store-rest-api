HTTP Routing Table

/api
 
/api
GET /api/v1/gifts/list List all gifts added to user's gift list.
GET /api/v1/gifts/list/report Produce report of purchased and non-purchased gifts within user list.
GET /api/v1/gifts/list/{gift_id} Retrieve gift information.
GET /api/v1/store/items List items method.
GET /api/v1/store/items/id/{item_id} Single item data retrieval method.
GET /api/v1/store/orders List orders retrieval method.
GET /api/v1/store/orders/id/{order_id} Order data retrieval method.
POST /api/v1/auth/login User authentication method.
POST /api/v1/auth/refresh User authentication refresh method.
POST /api/v1/auth/register User registration method.
POST /api/v1/gifts/list/add Add a gift to the user's gift list.
POST /api/v1/gifts/list/{gift_id}/purchase Purchase gift from list.
POST /api/v1/store/items Create a new item.
POST /api/v1/store/order Order creation method.
DELETE /api/v1/auth/logout User logout method.
DELETE /api/v1/auth/remove User removal method.
DELETE /api/v1/auth/revoke Revoke JWT refresh token.
DELETE /api/v1/gifts/list/{gift_id} Remove gift from user's gift list.
DELETE /api/v1/store/items/id/{item_id} Delete item from store. [Requires admin user]
DELETE /api/v1/store/orders/id/{order_id} Void order method.
